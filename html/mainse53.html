A.6 Gráficos

Para criar um esboço do gráfico de uma função de uma variável real , podemos usar a função plot. Esta função faz uma representação gráfica de pontos fornecidos. O Scilab oferece uma série de opções para esta função de forma que o usuário pode ajustar várias questões de visualização. Consulte sobre a função plot no manual do Scilab.

-->x = linspace(-2, 2, 100);  
-->plot(x, f(x)); xgrid
